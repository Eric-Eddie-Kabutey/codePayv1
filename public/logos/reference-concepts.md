# CodePay Logo Reference Concepts

All concepts follow Pomelo's minimalist, geometric sans-serif aesthetic while representing CodePay's African fintech infrastructure positioning.

## Concept 1: Geometric "C" Mark
**Style:** Modern geometric letterform (like Pomelo's approach)
- Bold, clean "C" in navy
- Minimalist and timeless
- Works at any size
- Similar geometric treatment to Pomelo's circular forms
- **Best for:** Professional, enterprise-focused positioning

## Concept 2: Connected Nodes "C"
**Style:** Geometric "C" with subtle connectivity dots
- "C" shape as main mark
- 3-4 small dots suggesting network/connectivity
- Represents infrastructure and growth
- Minimalist execution in navy with teal accents
- **Best for:** Emphasizes tech/infrastructure positioning

## Concept 3: African Continent "C"
**Style:** Geometric "C" with African map hint
- Clean "C" letterform
- Subtle African continent silhouette integrated
- Very subtle (not obvious at first glance)
- Premium, sophisticated feel
- **Best for:** Africa-first positioning without being heavy-handed

## Concept 4: Circuit Board "C"
**Style:** Geometric "C" with tech/blockchain elements
- "C" as primary shape
- Thin lines suggesting circuit patterns
- Blockchain/fintech aesthetic
- Minimal, elegant execution
- **Best for:** Tech-forward, infrastructure-focused message

## Concept 5: Pure Typography (Pomelo-style)
**Style:** Bold geometric sans-serif wordmark only
- "CodePay" in custom geometric sans
- Letter spacing and weight similar to Pomelo
- No icon mark, just typography
- Very clean and minimal
- **Best for:** Simplicity, scalability, modern fintech aesthetic

## Design Specifications

**Color Palette:**
- Primary: Navy #3d81b4 (main mark)
- Accent: Teal #14b8a6 (secondary elements)
- Background: White or transparent

**Scale Requirements:**
- Favicon (16px) ✓
- App icon (128px) ✓
- Navbar (40-48px) ✓
- Hero section (256px+) ✓
- Print (unlimited) ✓

**Style Match to Pomelo:**
- Geometric, not organic curves
- Bold and readable
- Modern sans-serif approach
- Minimal ornamentation
- Premium fintech aesthetic
- Clean lines, mathematical proportions
